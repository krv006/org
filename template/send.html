{% trans "Название статьи" %}

{% trans "Название статьи приводится английском языках. Оно должно быть информативным и раскрывать содержание статьи." %}

{% trans "Сведения об авторах" %}

Сведения об авторах, публикуемые на титульном листе статьи, включают следующие элементы: - Фамилия, Имя авторов; - полные названия организаций – мест работы авторов, полный почтовый адрес каждой организации (включая почтовый индекс). Допускается наличие нескольких аффилиаций; - адреса электронной почты авторов.

{% trans "Аннотация" %}

{% trans "Аннотация к статье представляется на русском и английском языках. Объем – 150–250 слов. Аннотация должна отражать основное содержание статьи и быть структурированной." %}

{% trans "Ключевые слова" %}

Ключевые слова приводятся на английском языке. Рекомендуемое количество ключевых слов (словосочетаний) – от 6 до 10. Ключевые слова или словосочетания отделяются друг от друга точками с запятыми.

{% trans "Введение" %}

{% trans "Во Введении даются ответы на вопросы почему и зачем было проведено исследование, какова исследовательская гипотеза." %}

{% trans "Основная часть" %}

Рекомендуется применять структуру IMRAD, предусматривающую, помимо Введения (Introduction), следующие разделы: «Методы» (Methods), «Результаты» (Results) и «Дискуссия» (Discussion).

{% trans "Заключение" %}

Заключение включает повторное обобщенное перечисление основных результатов исследования (в соответствии с заявленными целями, гипотезами и исследовательскими вопросами), а также оценку их значимости для науки. Также желательно указать направления дальнейших исследований в данной области. По объему Заключение, как правило, не превышает 10% от общего объема текста статьи.

{% trans "Литература" %}

Иностранная литература приводится на языке оригинала без сокращений. Список литературы оформляется в Гарвардском стиле. Для транслитерации русскоязычных/узбекоязычных наименований рекомендуется использовать существующие онлайн-сервисы. Рекомендуется, чтобы список литературы содержал не менее 15 источников. Элементы списка литературы нумеруются в порядке их упоминания (цитирования) в тексте статьи. Каждый из источников, включенных с список литературы, должен быть процитирован в тексте статьи хотя бы один раз. Цитирование источников в тексте статьи осуществляется по их номерам в списке литературы, в квадратных скобках (например, [2], [4; 5], [7–10], [3; 5; 7–10]).
